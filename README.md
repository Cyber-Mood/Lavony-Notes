# Lavony Notes 🌸

A simple, minimalistic and aesthetic note-taking web app built with pure HTML, CSS, and JavaScript.

## Features

- Create, edit, and delete notes  
- Pin important thoughts  
- Real-time search  
- Mood themes  
- Dark mode  
- Local storage saving  
- Subtle sparkle animation  

## How to Use

1. Clone the repository  
2. Open `index.html` in your browser  
3. Start writing  

No setup. No dependencies.

## Built With

- HTML
- CSS
- Vanilla JavaScript
